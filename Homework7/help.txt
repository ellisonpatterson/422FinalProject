Please type one of the following Commands:
   MainTester f {fileName}
   MainTester n {TestData object}
   MainTester
  
 f: 
 The thing following f is the name of the text file containing the attribute values. 
 An example file name is "testData.txt". The data inside the file is all on one line and
 is separated by a pipe, "|".  The data types expected are as follows:
   int|String|String|int|int|int|int|String|String|String|String|String
 
 n:
 The things following n is the test object. The object must be of type TestData.
 
 If you do not enter a command after MainTester, the default data will be used to run
 the tests.
